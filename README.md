# OneText-Library_Honkai

收集一些出自《崩坏3》内的颇有感想的句子（

包括但不限于该系列的 **主线剧情 / 活动剧情 / 生日剧情 / 视觉小说 / 漫画。**

排版原因，可能对部分文字进行修改。

**适用于[一言 OneText](https://github.com/lz233/OneText_For_Android) | [官方句子库](https://github.com/lz233/OneText-Library)**

	"from_key": "from"
}
